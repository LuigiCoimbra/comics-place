## Visualizar
 - No seu terminal, clone o respositório:
   - ``` git clone git@github.com:marcusrodriguesdev/comic-store.git ```
 - Entre no diretório do projeto
   - ``` cd comic-store```
 - Na pasta raíz, instale as dependências:
   - ``` npm install ```
 - E inicialize
   - ``` npm start ```
